# Sales Performance Dashboard

An interactive sales analytics dashboard built with HTML, CSS, and Chart.js.

## Features
- Monthly revenue & profit trend
- Revenue by region (donut chart)
- Product revenue breakdown
- Online vs Offline margin comparison
- Volume vs Value bubble chart
- Region × Product profit margin heatmap
